Realtime multiplayer games made easy.

Easily program and deploy your own socket.io server using visual code
all on your iPad for free!

Simple Flowchart Visuals

Inspired by and created with hyperPad, CSocket uses simple flowcharts to represent code. Drag and drop snippets and create complex algorithms in seconds.

Advanced Modules

CSocket includes advanced features such as loops, conditionals, timers, functions, memory-analytics, HTTP Requests, console inputs and many more.

Deploy within Minutes

Quickly compile and deploy your socket.io server with ease all on your iPad using GitHub and Heroku. No expensive software required and 100% computer-free.

How to use

• Download hyperPad OR hyperPad viewer from the App Store.
• Sign up for GitHub and Heroku.
• Open the project via hyperPad or hyperPad viewer.
